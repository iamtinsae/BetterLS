## :wrench: <samp>SETUP</samp>

   This is step-by-step how to install and use `bls`. 
   <br />
   > Make sure you have python3 installed!
   ```sh

    > cd BetterLS && chmod +x ./bls.py
    > ./bls.py /
   ```
